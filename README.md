# LoginApp

LoginApp is a simple application that allows users to register and log in using a username and password. The application is written in Java.

## Features

- User Registration
- User Login
- Interactive Menu

## Prerequisites

- Java Development Kit (JDK)

## How to Run

1. **Clone the repository:**

    ```bash
    git clone https://github.com/username/LoginApp.git
    cd LoginApp
    ```

2. **Compile and run the program:**

    ```bash
    javac LoginApp.java
    java LoginApp
    ```

## Project Structure

- `User.java`: Class for the user object.
- `UserService.java`: Class for managing users.
- `LoginApp.java`: Main class that runs the application.

## Usage Example

1. **User Registration:**

    ```plaintext
    1.Register
    2.Login
    3.Exit
    1
    Enter Your Username: Alice
    Enter Your Password: 1234
    Registration Successful
    ```

2. **User Login:**

    ```plaintext
    1.Register
    2.Login
    3.Exit
    2
    Enter Your Username: Alice
    Enter Your Password: 1234
    Login Success
    Hello Alice
    ```

3. **Exit the Application:**

    ```plaintext
    1.Register
    2.Login
    3.Exit
    3
    ```
